# 🖥️ Bytesweep - Tech Store Built with React

Bytesweep is a modern, responsive e-commerce website for electronics and tech gadgets, built using 'React.js'. This project demonstrates core frontend development practices with clean UI and component-based architecture.

# 🚀 Features

✅ Fully Responsive Design
✅ Built with React + Vite for fast performance
✅ Clean, modern UI with Tailwind CSS (Optional)
✅ Interactive Carousel Banner
✅ Product Cards for Featured Items
✅ Reusable Navbar Component
✅ Routing ready for future expansion (React Router recommended)
✅ Easy to Customize and Extend

---


# 🛠️ Tech Stack

* React.js
*  Vite (for faster development)
* JavaScript (ES6+)
* Tailwind CSS (as per your project setup)
* HTML5

# 💡 Getting Started

# 1️⃣ Clone the Repository

```bash
git clone https://github.com/sarthakmahapatra05/BYTESWEEP.git
```

Navigate to the project:

```bash
cd BYTESWEEP/Desktop/Bytesweep\ in\ react
```

# 2️⃣ Install Dependencies

```bash
npm install
```

# 3️⃣ Run the Development Server

```bash
npm run dev
```

The project should now be live at [http://localhost:5173](http://localhost:5173) (Vite default port).
